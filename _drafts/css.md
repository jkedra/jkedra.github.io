### side notes
{% highlight css %}
p.note {
    border-style: solid;
    border-width: 1px;
    border-left-width: 15px;
    padding: 10px;
}
{% endhighlight %}

<p class="note"><b>Note:</b>
    The "border-left-width" property does not work if it is used alone.
    Use the "border-style" property to set the borders first.
</p>
